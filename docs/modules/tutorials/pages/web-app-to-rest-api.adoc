==== Scenario example
For example we need to verify that all resources by CSS selector `a` of web application `https://example.com` are valid. We know web app has sitemap with relative URL `/sitemap.xml`, but also know that some of the pages have similar redirections.

Then we should add additional property:

.Properties
[source,gherkin]
----
web-application.main-page-ur=https://example.com
transformer.from-sitemap.filter-redirects=true
----

.Usage example
[source,gherkin]
----
Scenario: Verification of all resources on all the pages using Sitemap
Then all resources by selector `a` are valid on:
|pages     |
|<page-url>|
Examples:
{transformer=FROM_SITEMAP, siteMapRelativeUrl=/sitemap.xml, ignoreErrors=true, column=page-url}
----
