# Added

### #6554 [vividus-plugin-rabbitmq] Add RabbitMQ plugin
New plugin with steps to publish and retrieve messages from RabbitMQ brokers using configurable connections.

### #6573 [vividus-plugin-rabbitmq] Add ability to use SSL for RabbitMQ connections
New property:
```properties
rabbitmq.<broker-key>.use-ssl=false
```
Enables encrypted (TLS) connections to RabbitMQ brokers.

### #6590 [vividus-plugin-rabbitmq] Add ability to set RabbitMQ message properties
New step:
```gherkin
When I set RabbitMQ message properties:$properties
```
Allows setting AMQP message properties (e.g. `contentType`, `correlationId`, `messageId`, `appId`) for the next published message.

### #6574 [vividus] Add step to wait for file appearance in directory
New step:
```gherkin
When I wait `$timeout` with `$pollingTimeout` polling until file matching `$fileNameRegex` appears in directory `$directoryPath` and save path to $scopes variable `$variableName`
```
Polls a directory until exactly one matching file appears and saves its absolute path to a variable.

# Changed

### #6532 [vividus-plugin-json] Improve `FROM_JSON` transformer columns parsing
`columns` parameter now supports multiline formatting with semicolon-separated column names, improving readability of large column lists.

### #6589 [vividus-plugin-mobile-app] Add logging for app activation step
The step activating an application by bundle identifier now logs the bundle identifier for better traceability.

# Fixed

### #6583 [vividus] Fix `statistics.json` not counting stories cancelled by timeout
Timed-out cancelled stories are now correctly included in generated execution statistics.

### #6584 [vividus-extension-selenium] Fix capturing of HTTP response with missing fields
Fixed failures in HTTP response capture flow when expected response fields are missing.

### #6602 [vividus-plugin-visual] Resolve PMD configuration issue for `blobStorage` feature
Fixed PMD setup for the `blobStorage` source set to prevent CI failures in visual plugin checks.
